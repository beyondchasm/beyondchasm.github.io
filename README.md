## Project Scaffolding (Obsidian + Quartz)

프로젝트 표준 폴더/문서를 자동 생성합니다.

명령:

`npm run create-project -- wordcard`

옵션 예시:

`npm run create-project -- wordcard --name "WordCard" --stack spring --stack postgres --tag app`

## Blog Scaffolding (Obsidian + Quartz)

블로그 글 템플릿을 생성하고 `content/blog/index.md`에 자동 등록합니다.

명령:

`npm run create-article -- why-quartz --title "왜 Quartz를 쓰는가" --tag quartz --tag obsidian`
